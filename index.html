Authentification

L'authentification utilise les sessions Laravel sans JWT. Le middleware CheckAuth protège toutes les routes privées.

AuthController.php — login()
$utilisateur = Utilisateur::where('nom', $request->nom)->first();

if ($utilisateur && $utilisateur->checkPassword($request->mdp)) {
    Session::put('user_id', $utilisateur->id);
    Session::put('user_nom', $utilisateur->nom);
    Session::put('is_admin', $utilisateur->isAdmin());
    return redirect()->route('praticiens.index');
}
return back()->withErrors(['login' => 'Identifiants incorrects.']);
Utilisateur.php — checkPassword() — SHA-256 + HMAC
public function checkPassword($password)
{
    // Vérification HMAC SHA-256
    $hashedPassword = hash_hmac('sha256', $password, config('app.key'));
    if ($this->mdp === $hashedPassword) return true;

    // Vérification SHA-256 simple
    $simpleHash = hash('sha256', $password);
    if ($this->mdp === $simpleHash) return true;

    return false;
}
CheckAuth.php — Middleware
if (!Session::has('user_id')) {
    return redirect()->route('login')
        ->with('error', 'Veuillez vous connecter pour accéder à cette page.');
}
return $next($request);

La mise à jour de la date d'embauche déclenche un trigger MySQL qui recalcule automatiquement l'ancienneté, l'échelon et le salaire via la table grille_salariale. Le formulaire n'est visible que pour les admins.

PraticienController.php — updateAnciennete()
$praticien = Praticien::findOrFail($id);
$praticien->date_embauche = $request->date_embauche;

ApiService — getPraticiens()
static Future<List<Praticien>> getPraticiens({String sort = 'nom'}) async {
  final res = await http.get(
    Uri.parse('$BASE_URL/praticiens?sort=$sort'),
    headers: {'Accept': 'application/json'},
  );
  if (res.statusCode == 200) {
    final data = jsonDecode(res.body);
    return (data['data'] as List).map((e) => Praticien.fromJson(e)).toList();
  }
  throw Exception('Erreur serveur (${res.statusCode})');
}

void _search(String q) {
  setState(() {
    _filtered = q.isEmpty
        ? _praticiens
        : _praticiens.where((p) =>
              p.nomComplet.toLowerCase().contains(q.toLowerCase()) ||
              p.type.toLowerCase().contains(q.toLowerCase()))

  });
}
